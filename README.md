Overview

This project is a simple News Explorer Application built using UIKit, focusing on clean architecture and native performance without relying on third-party dependencies.
Tech Stack & Architecture

* Framework: UIKit
* Architecture: MVVM (Model–View–ViewModel)
* Language: Swift

Image Handling
* Implemented image downloading and caching using native iOS APIs.
* No third-party libraries were used for image loading or caching.

  
Development Notes


* The project emphasizes simplicity, performance, and maintainability.
* Native solutions were preferred to keep the app lightweight and dependency-free.

  
Time Estimation


* Estimated development time: ⏱️ 2 hours 30 minutes
